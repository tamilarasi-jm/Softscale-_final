# Scale Cost Calculator

A modern web application for calculating and visualizing network scaling costs using React and TypeScript. This tool helps users understand the cost implications of scaling their network infrastructure by providing an interactive diagram and detailed cost breakdowns.

## Features

- Interactive network diagram with draggable nodes
- Real-time cost calculations
- Responsive design that works on desktop and mobile
- Exportable cost reports
- Customizable network configurations

## Getting Started

### Prerequisites

- Node.js 16+ and npm 8+ (recommend using [nvm](https://github.com/nvm-sh/nvm#installing-and-updating))

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/your-username/scale-cost-calculator.git
   cd scale-cost-calculator
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build the app for production
- `npm run preview` - Preview the production build locally
- `npm run lint` - Run ESLint
- `npm run type-check` - Run TypeScript type checking
